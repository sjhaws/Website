# Nephi Journey

A small Phaser-based retro platformer prototype themed around Nephi's journey.

## How to run

- Open `index.html` in a browser.
- If your browser blocks local script loading, serve the folder with any simple static web server.

## Controls

- Left and right arrows or `A` / `D`
- Jump with `Up`, `W`, or `Space`
- Touch screen buttons are shown on mobile or tablet screens

## Gameplay

- There are 6 levels.
- Reach the far right edge of each 5000px level to advance.
- Touching an enemy restarts the current level.
- Scroll pickups grant 5 seconds of invincibility.
